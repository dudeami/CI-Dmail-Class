Alternative email class for Codeigniter. Based off the original Email class.

Codeigniter Download: http://bitbucket.org/ellislab/codeigniter/

Requirements:
 - Codeigniter 2
 - PHP5
 
Installing:
 - Create the folder application/libraries/Dmail/
 - Place the Dmail.php file and drivers folder into the created folder

Usage:
 - Include both the driver and dmail library (In that order)
 - Use as you would with the original Email class
